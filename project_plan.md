# Đà Lạt Home Rental - Trang Web Bất Động Sản Đà Lạt

## 1. Mô Tả Dự Án
Trang web bất động sản chuyên biệt cho thị trường Đà Lạt, bao gồm: cho thuê nhà nguyên căn, homestay và mua bán căn hộ. Hướng đến khách hàng muốn tìm kiếm bất động sản tại Đà Lạt với trải nghiệm dễ dàng và trực quan.

## 2. Cấu Trúc Trang
- `/` - Trang Chủ (Home)
- `/search` - Tìm Kiếm Bất Động Sản (Search Listings)
- `/homestay` - Cho Thuê Homestay
- `/apartment` - Mua Bán Căn Hộ
- `/map` - Bản Đồ
- `/contact` - Liên Hệ

## 3. Tính Năng Cốt Lõi
- [x] Trang chủ với hero, tìm kiếm nhanh, featured listings
- [x] Tìm kiếm theo khu vực, loại hình, giá
- [x] Danh sách cho thuê nhà nguyên căn
- [x] Danh sách homestay
- [x] Danh sách mua bán căn hộ

## 4. Tích Hợp Bên Thứ Ba
- Supabase: Chưa cần (dùng mock data)
- Google Maps: Nhúng iframe embed
- Form: Readdy Form API

## 5. Kế Hoạch Phát Triển

### Phase 1: Xây Dựng Toàn Bộ Website (Hiện tại)
- Mục tiêu: Hoàn thiện 6 trang website với đầy đủ UI/UX
- Deliverable: Website hoàn chỉnh với mock data

### Phase 2: Kết Nối Dữ Liệu Thực (Tương lai)
- Mục tiêu: Kết nối Supabase để quản lý listings thực
- Deliverable: Admin panel, CRUD cho bất động sản
